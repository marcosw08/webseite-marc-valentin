Was ist Bootstrap eigentlich?

Bootstrap hat sich einen Namen gemacht. Indem sie ihre Gestaltung verkaufen. Gratis. Zumindest ein Teil. Bootstrap hat auf ihrer Webseite ganz viele Beispiele, die man als HTML ins Html kopieren kann. Das CSS ist dan extern auf einem Server gespeichert. Den grossen Namen hat sich Bootstrap gemacht, da sehr viele Firmen Bootstrap benutzen. Ich finde unsere Webseite sieht der Apple-Webseite sehr ähnlich. Ich hoffe natürlich nicht, dass Apple mit Bootstrap gemacht ist. Es ist eine wirklich gute Arbeit von den Leuten von Bootstrap.
